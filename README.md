# Fortress Vault 2.0 - Zero-Knowledge Password Manager

Welcome to Fortress Vault, a local password manager built with robust encryption (256-bit AES-GCM), developed in Python and powered by a PyQt6 graphical user interface.

- About Me & Project Philosophy

I am a passionate tech enthusiast currently studying computer science theory through traditional textbooks. However, I firmly believe that hands-on, practical experience is infinitely more valuable than theory alone when it comes to truly understanding how software is built.

To put this belief into practice, I dedicate my free time to developing real-world applications. In order to bridge the gap between academic theory and the practical complexities of coding, I collaborate with Artificial Intelligence, leveraging it as both an interactive tutor and a development "co-pilot."

This repository is the direct result of that experiment: a convergence of traditional study, a practical "learning-by-doing" approach, and the power of AI.

- Project Status

The software is currently in its early development stage (Alpha). While the cryptographic core and underlying architecture are stable and fully operational, there is still significant work to be done, particularly on two fronts:

UI/UX (Graphics): The current user interface is quite minimal and spartan. My goal is to make it look considerably more modern, fluid, and visually engaging.
Missing Features: While the application currently supports the creation of new categories, essential features—such as deleting categories, modifying existing records, and a more dynamic database management system—have yet to be implemented.

Tech Stack

- Language: Python 3.13+
- GUI Framework: PyQt6
- Cryptography: cryptography library (AES-256-GCM authenticated encryption)
- Storage: Locally encrypted JSON database
